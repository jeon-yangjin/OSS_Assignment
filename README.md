# ­ЪќЦ№ИЈ вфЁва╣Вќ┤ ВёцвфЁ

### 1. top
- ВІюВіцьЁюВЮў ВІцВІюЖ░ё ьћёвАюВёИВіцВЎђ ВъљВЏљ ВѓгВџЕ ьўёьЎЕВЮё в│┤ВЌгВцї
- CPU, вЕћвфевдг ВѓгВџЕвЪЅ ьЎЋВЮИ Ж░ђвіЦ
- ВбЁвБї: `q`

### 2. ps
- ьўёВъг ВІцьќЅ ВцЉВЮИ ьћёвАюВёИВіц вфЕвАЮ ВХюваЦ
- ВБ╝Вџћ ВўхВЁў:
  - `ps -e` : вфевЊа ьћёвАюВёИВіц ьЉюВІю
  - `ps aux` : ВЃЂВёИ ВаЋв│┤ ьЉюВІю

### 3. jobs
- ьўёВъг ВЁИВЌљВёю ВІцьќЅ ВцЉВЮИ ВъЉВЌЁ вфЕвАЮ ьЎЋВЮИ
- в░▒ЖиИвЮ╝Вџ┤вЊю ВъЉВЌЁ Ж┤ђвдгВЌљ ВюаВџЕ
- ВўѕВІю: `jobs -l`

### 4. kill
- ьћёвАюВёИВіц ВбЁвБї вфЁва╣
- ВѓгВџЕв▓Ћ: `kill [PID]`
- Ж░ЋВаю ВбЁвБї: `kill -9 [PID]`
